## Simple REST API service for saving birthdays

using Gin and Gorn

How to use: 
```$ go run main.go```

### endpoints
Create birthday
```POST /birtdays```

Get all birthdays
```GET /birtdays```

Get birthday by ID
```GET /birtdays/{id}```

Update birthday 
```PATCH /birtdays/{id}```

Delete birthday
```DELETE /birtdays/{id}```
